# Nokia Snake Codex

A simple browser-based Snake game inspired by the classic Nokia phone version.

## Run

Open `index.html` in any modern web browser.

## Controls

- Arrow keys to move
- `W`, `A`, `S`, `D` to move
- `Space` to restart after game over
- On-screen buttons for touch play

## Features

- Retro green-screen style look
- Score and best-score tracking
- Increasing speed as the snake grows
- Works as a single standalone HTML file
